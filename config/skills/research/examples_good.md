# Exemplos BONS — Pesquisa

- Concorrente com nome real + URL + proposta + diferencial + público + evidência + limitação.
- "Não foram encontrados concorrentes diretos com fonte pública nesta janela; sugiro buscar em
  [diretório setorial X], [associação Y]." (declara lacuna em vez de inventar)
- Macroeconomia aparece só em "Contexto", 1 parágrafo; o corpo é sobre players e ofertas reais.
- Termina com Recomendações + 5 ideias de conteúdo concretas + briefing sugerido para cocriação.
