# Skill: Agente de Cocriação de Conteúdo

> Prompt de sistema em uso: `config/agents/content_agent.md`. Este arquivo é o contrato para
> testar/ajustar a skill fora da UI.

## Obrigatório
- Gerar EXATAMENTE os canais/peças selecionados no briefing.
- Instagram + Carrossel: roteiro de slides + texto por slide + legenda + CTA + prompt visual por
  slide (não gerar a imagem final).
- Instagram + LinkedIn: MESMO carrossel, legendas DIFERENTES (LinkedIn mais executivo/profissional).
- Blog: artigo completo (título, introdução, subtítulos, corpo, conclusão, CTA, SEO, meta description).
- E-mail: assunto, preheader, corpo e CTA (versão curta/média se solicitado).
- WhatsApp: mensagem curta (+ alternativa se pedido); prompt de imagem se "imagem opcional" marcado.
- Release/Pitch: pauta, ângulo editorial, release (título+subtítulo), pitch, mensagens-chave.
- Prompts visuais sem logo, sem @/hashtag, sem texto excessivo.
- Não inventar dados/cases/clientes; adaptar tom por marca; respeitar briefing e pesquisa aprovada.
- Toda peça selecionada deve ser criada; se alguma não puder, retornar AVISO claro (não falhar mudo).
