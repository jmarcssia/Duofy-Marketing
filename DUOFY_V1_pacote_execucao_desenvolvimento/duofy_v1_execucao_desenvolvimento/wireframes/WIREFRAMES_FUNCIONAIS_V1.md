# Wireframes Funcionais — DUOFY V1

## Layout geral

- Sidebar fixa à esquerda.
- Topbar com filtro global por marca.
- Conteúdo principal em cards.
- Ações primárias sempre visíveis.
- Linguagem de negócio, não técnica.

## Dashboard

Cards superiores:

1. Tarefas em aberto.
2. Outputs em revisão.
3. Conteúdos aprovados.
4. Custo do período.
5. Tokens usados.

Blocos:

- Atividades recentes.
- Próximos eventos do calendário.
- Resumo do orquestrador.
- Uso por agente.

## Chat / Orquestrador

Colunas:

1. Sessões à esquerda.
2. Chat no centro.
3. Contexto/tarefas à direita.

Ações rápidas:

- Nova pesquisa.
- Criar conteúdo.
- Criar calendário.
- Criar pauta/release.
- Consultar memória.

## Co-criação

Topo:

- Marca.
- Canal.
- Formato.
- Objetivo.
- Persona.

Abas:

- Legenda.
- Arte.
- Carrossel.
- Reels.
- LinkedIn.
- Blog/E-mail.
- Webinar.
- Prompts visuais.

Botões:

- Gerar.
- Gerar nova versão.
- Salvar rascunho.
- Enviar para aprovação.
- Copiar.
- Exportar briefing.

## Pesquisa

Topo:

- Tema.
- Marca.
- Profundidade.
- Fontes.
- Rodar agora.

Resultado:

- Resumo executivo.
- Fontes.
- Sinais.
- Oportunidades.
- Recomendações.
- Enviar para co-criação.
- Enviar para assessoria.
- Salvar como memória.

## Aprovações

Lista à esquerda:

- Status.
- Marca.
- Agente.
- Data.

Preview/editor à direita:

- Conteúdo.
- Histórico de versões.
- Comentários.

Botões:

- Aprovar.
- Solicitar ajuste.
- Rejeitar.
- Arquivar.
- Exportar.

## Memória / Documentos

Topo:

- Upload.
- Busca.
- Filtros.

Lista:

- Documento.
- Marca.
- Categoria.
- Status.
- Chunks.

Detalhe:

- Resumo.
- Tags.
- Chunks.
- Memórias relacionadas.

## Custos e Tokens

Cards:

- Custo total.
- Tokens totais.
- Custo por agente.
- Custo por marca.
- Modelo mais usado.

Gráficos:

- Uso por dia.
- Uso por modelo.
